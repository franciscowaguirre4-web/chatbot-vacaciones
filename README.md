# Chatbot para Solicitud de Vacaciones

## Descripción

Este proyecto corresponde al Trabajo Integrador de Organización Empresarial.

El chatbot permite:

- Consultar los días de vacaciones disponibles.
- Validar el legajo del empleado.
- Solicitar días de vacaciones.
- Aprobar o rechazar la solicitud.
- Mostrar mensajes de error cuando el usuario ingresa datos inválidos.

## Tecnologías utilizadas

- Python
- CSV
- GitHub

## Archivos

- chatbot.py
- empleados.csv
- diagrama_bpmn.png

## Cómo ejecutar

1. Descargar el proyecto.
2. Abrir la carpeta en Visual Studio Code.
3. Ejecutar:

```bash
python chatbot.py
```
